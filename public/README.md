Drop these two files here before deploying:

- `photo.jpg` — Shaurya's professional portrait (portrait orientation, ~800×1000+).
  Until present, the Lore section renders a styled "SV" monogram placeholder
  automatically (see `components/ui/Photo.tsx`). The build does not break without it.

- `resume.pdf` — Latest resume. The Hero and Contact "Download Resume" buttons
  link to `/resume.pdf` and download it as `Shaurya_Verma_Resume.pdf`.
